Description:
---
>>  A simple website with samurai heart

How to start:
---
*   复制.env文件,并根据自身环境修改参数
*   执行`composer install -vv`
*   执行`php artisan key:generate`
*   执行`php artisan serve`,通过127.0.0.1:8000或者通过本地环境/owarai/public进入项目
